## üèóÔ∏è Flagship Project ‚Äì ITIL √ó AWS 3-Tier SmartSolve Portal

**Repo:** [`itil-aws-sla-3tier-smartsolve-portal`](https://github.com/jsamuelkamau-dot/itil-aws-sla-3tier-smartsolve-portal)

A **3-tier IT support portal** designed to show how ITIL practices map onto real AWS architecture and SLAs.

**Architecture highlights:**

- **Network & Security**
  - Dedicated **VPC** with public subnets for ALB/EC2 and private subnets for the database.
  - **Security groups** enforcing least privilege between web, app and data tiers.
  - Internet access only via **Application Load Balancer (ALB)**.

- **Compute & Data**
  - Highly-available **EC2 web/app tier** across multiple AZs.
  - **RDS** database in private subnets.
  - Support for components like **ECS / EKS / Lambda / S3 / IAM / CloudWatch** as the project grows.

- **ITIL & SLA Mapping**
  - Designed to demonstrate **Incident Management, Change Management and Service Level Management**.
  - Use of **CloudWatch metrics, logs and alarms** to simulate SLA monitoring (availability, response times, error rates).
  - Documentation and LinkedIn posts describing **Day-by-Day build logs** (VPC ‚Üí subnets ‚Üí routing ‚Üí ALB ‚Üí EC2 ‚Üí monitoring).

This project is my main portfolio piece to show how I think about **operations, reliability and security**, not just spinning up random AWS services.
